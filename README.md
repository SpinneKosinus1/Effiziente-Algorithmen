## About

Dieser Repository beinhaltet für den Kurs "Effiziente Algorithmen" alle relevanten Dokumentationen, Zusammenfassungen und Übungsaufgaben inkl. Prüfungsvorbereitungen.

## Aufbau & Struktur

Das Repository besitzt folgenden Struktur

- **notebooks** - Zusammenfassungen, Erklärungen und Analysen
- **assignments** - Übungsaufgaben und Prüfungsvorbereitungen
- **examples** - Demos, Prototypen und Showcases
- **images** - Bilder und Zeichnungen
- **scripts** - Wrappers, Tools, Hilfsfunktionen usw.
- **files** - Sonstige Dateien

Die Dateien werden dabei im Jupyter Notebook Format bereitgestellt. Zur Verwendung sollten die bereitgestellten Anforderungen installiert werden.
